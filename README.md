# MikaNote

<img width="2557" height="1386" alt="스크린샷 2026-04-07 111510" src="https://github.com/user-attachments/assets/457e30d5-b60b-4b77-8cdf-c2d402437c01" />

MikaNote is a desktop sticky memo and todo note app for Windows.

## Features
- Sticky memo notes
- Todo notes
- Desktop context menu creation
- Hide / Trash note management
- Tray icon support
- Windows startup support

## Installation
Download the latest `MikaNote-Setup.exe` from the Releases page.

## Requirements
- Windows 10/11 x64
- .NET 8 Desktop Runtime is installed automatically by the setup
